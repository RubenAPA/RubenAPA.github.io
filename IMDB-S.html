IMDB DATASET

Área: Limpieza y Análisis de Datos.
Creado por: Rubén Padilla Aguilar.
Publicado en: 15-12-2023.
Herramientas utilizadas: AZ Blob Storage, AZ Databricks y Power BI.
Conceptos utilizados: Mounting, PySpark, Scala, Delta Tables, DAX.

Información del conjunto de datos:
1. El conjunto de datos se descargó de Kaggle bajo el nombre "Messy IMDB dataset" por David Fuente Herraiz.
2. Tiene 101 registros y 12 columnas.
3. También presentaba problemas de calidad de datos, incluidos valores nulos, formatos incorrectos, convenciones de nombres inconsistentes y valores no coincidentes.

Problema empresarial:
En un escenario hipotético, una empresa tiene un archivo con datos sucios en AZ Blob Storage y necesitan estos datos específicos para la creación de un informe centrado en los detalles de cada película y director, para analizar las tendencias relacionadas con toda la información de las películas. Para completar esta tarea, desean limpiar los datos utilizando Databricks porque han usado notebooks para limpiar sus datos en su proceso ETL, por lo que es una herramienta que ya han utilizado. Después de las transformaciones, los datos deben conectarse a Power BI para la creación del reporte.

Objetivos:
1. Limpiar los datos para adquirir experiencia utilizando múltiples funciones en PySpark.
2. Mejorar la comprensión de las limitaciones y capacidades de Databricks para la transformación de datos.
3. Desarrollar un informe para facilitar la comprensión de las tendencias en los datos, y también aplicar nuevas características como los Tooltips.

IMDB DATASET

Domain: Data Cleansing and Analysis.
Created by: Rubén Padilla Aguilar.
Posted on: 12-15-2023.
Tools used: AZ Blob Storage, AZ Databricks and Power BI.
Concepts used: Mounting, PySpark, Scala, Delta Tables, DAX.

Business problem:
In a hypothetical scenario, a company has a file with dirty data in Azure Blob Storage, they need this specific data for the creation of a report focused on the details of each movie and director, so they could analyze the trends related to the whole movies information. To complete this task, they want to clean the data using Databricks because they have used notebooks to clean their data in their ETL process, so it’s a tool that they already used. After the transformations the data need to be connected to Power BI for the creation of the report.

Dataset Info:
1. The dataset was downloaded from Kaggle under the name "Messy IMDB dataset" by David Fuente Herraiz.
2. Has 101 records and 12 columns.
3. Also, exhibited data quality issues, including null values, incorrect formats, inconsistent naming conventions, and mismatched values.

Objectives:
1. Clean the data to acquire experience using multiple functions in PySpark.
2. Improve the understanding of limitations and capabilities of Databricks for the transformation of data.
3. Develop a report to facilitate the comprehension of trends in the data, and also apply new features like the Tooltips.
